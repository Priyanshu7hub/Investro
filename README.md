🚀 Investro – Smart Investment Manager

Investro is a full-stack web application that helps users manage, track, and analyze their investments in a simple and efficient way. It provides a clean dashboard to monitor financial growth and make smarter investment decisions.

📌 Key Features
📊 Dashboard Overview – View all investments in one place
💼 Portfolio Management – Add, update, and delete assets
📈 Performance Tracking – Analyze profit/loss trends
🔐 User Authentication – Secure login & signup system
⚡ Responsive UI – Works smoothly on all devices

⚙️ Installation & Setup
1️⃣ Clone the Repository
git clone https://github.com/Priyanshu7hub/Investro.git
2️⃣ Go to Project Directory
cd Investro
3️⃣ Install Dependencies

For backend

cd server
npm install

For frontend

cd client
npm install
4️⃣ Setup Environment Variables

Create a .env file in the server folder:

PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
5️⃣ Run the Project

Backend

npm start

Frontend

npm start
